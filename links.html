Classes Taken at CSU

  • CSCI 235 Procedural Programming. This class gave me the foundational knowledge of C++ programming.
  • CSCI 332 Applied Networking. This class gave me the foundational knowedge of computer networks and how to apply this knowledge in real world situations. This class also gave me the knowledge to build this website using HTML and CSS to promote the knowledge I gained in school.
  • CSCI 301 Survey of Scripting Languages. This class allowed me to use and explore different programming languages such as Java, Python, PHP, and more. I was able to see first hand the value of automation by writing different scripts.
  • CSCI 433 Network Security. This class taught me the importance of best practices when it comes to staying safe and secure online. I learned many techniques that hackers use to steal information and just as many techniques to fight back by staying ahead of the hackers.
  • CSCI 352 Cyber Defense. This class helped to teach me hands on skills with defending computers by configuring firewalls and how intrusion detection and prevention systems are implemented.
  • CSCI Principles of Cybersecurity. This class taught me best practices of computers and networks. This class was less technical and more thought provoking on how to train people to stay safe online.
  • CSCI 210 Intro to Computer Science Through Robotics. This class was very hands on by creating code to program a small robot to move in a sequence of my choosing.
